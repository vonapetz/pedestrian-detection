# 🚶 Обнаружение пешеходов на видео

## Описание

Этот проект предназначен для автоматической детекции и отрисовки людей (пешеходов) на видео с помощью современных моделей YOLO (YOLOv8, YOLO11). Входное видео анализируется покадрово, все обнаруженные пешеходы выделяются на каждом кадре прямоугольниками с указанием класса и уровня конфиденса.

## ✨ Возможности

- Детекция людей на видео с разной дистанцией до объекта
- Отрисовка bounding boxes с классом и уверенностью на каждом кадре
- Корректная работа на Windows, Linux, macOS
- Поддержка быстрого инференса для больших видео
- Гибкий выбор модели (YOLOv8, YOLO11 — nano, small, medium, large, xlarge)
- Вывод статистики качества и производительности

## 🛠 Установка

1. Клонируйте репозиторий проекта:
git clone https://github.com/vonapetz/pedestrian-detection.git


2. Перейдите в папку проекта:
cd pedestrian-detection


3. Установите зависимости:
pip install -r requirements.txt


## ✅ Быстрый старт

1. Поместите ваше видео (например, `crowd.mp4`) в папку `data/`.

2. Запустите обработку:
python main.py --input data/crowd.mp4 --output output/result.mp4

3. Дополнительные параметры:
- `--model` — выбор варианта модели (по умолчанию `yolo11n.pt`)
- `--confidence` — минимальный порог уверенности (`0.0-1.0`)
- `--device` — устройство для инференса (`cpu`, `cuda`)
- `--imgsz` — устройство дл изменения размера входного изображения (по умолчанию `640`)

Например, для работы на GPU и большей точности:
python main.py --input data/crowd.mp4 --output output/result2.mp4 --model yolo11x.pt --confidence 0.8 --device cuda --imgsz 1920
